# DIASMATH

Plataforma de tecnologia educacional para transformar o ensino da Matemática.

## Módulos iniciais

- Play: jogos educativos.
- Labs: laboratórios digitais.
- Professor: turmas e acompanhamento.
- Aluno: jornada gamificada.
- Escola: indicadores institucionais.

## Como rodar localmente

```bash
npm install
npm run dev
```

Depois acesse:

```bash
http://localhost:3000
```

## Publicação

1. Enviar este projeto para o GitHub.
2. Importar o repositório na Vercel.
3. Conectar o domínio diasmath.com.br.
4. Configurar Supabase futuramente.
