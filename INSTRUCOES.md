# Atualização 16 — Contato oficial DIASMATH™

Esta atualização coloca o e-mail oficial no site:

```text
contato@diasmath.com.br
```

## Arquivos incluídos

```text
components/Header.tsx
components/Footer.tsx
app/contato/page.tsx
```

## O que muda

1. O menu superior ganha a aba `Contato`.
2. O botão azul do topo passa a abrir e-mail para `contato@diasmath.com.br`.
3. O rodapé passa a mostrar `contato@diasmath.com.br`.
4. O site ganha a página `/contato`.

## Como publicar no GitHub

1. Baixe o ZIP.
2. Extraia o ZIP.
3. Abra o repositório `diasmath-platform`.
4. Clique em `Add file`.
5. Clique em `Upload files`.
6. Envie as pastas extraídas:

```text
components
app
```

7. Em `Commit message`, escreva:

```text
Adiciona contato oficial da DIASMATH
```

8. Clique em `Commit changes`.
9. Aguarde a Vercel publicar.
