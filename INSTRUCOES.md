# Atualização 06 — DIASMATH LABS: GeoTessela Lab

## Como subir no GitHub

1. Extraia este ZIP.
2. Abra o repositório `diasmath-platform`.
3. Clique em `Add file` → `Upload files`.
4. Arraste as pastas:
   - `public`
   - `app`
5. Aguarde o upload.
6. Clique em `Commit changes`.
7. Aguarde o deploy automático da Vercel.
