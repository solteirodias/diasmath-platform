# Próximos passos da DIASMATH

## 1. Enviar para o GitHub

Crie um repositório chamado:

diasmath-platform

Depois envie estes arquivos.

## 2. Importar na Vercel

Na Vercel, clique em:

Add New Project

Escolha o repositório do GitHub.

## 3. Publicar

A Vercel vai detectar Next.js automaticamente.

## 4. Conectar domínio

Depois de publicar, vá em:

Settings > Domains

Adicione:

diasmath.com.br

A Vercel vai mostrar os DNS que você precisa configurar no Registro.br.

## 5. Próxima etapa técnica

Conectar Supabase para login, cadastro e banco de dados.
