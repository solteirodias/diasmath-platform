GeoTessela Lab

Laboratório digital de Geometria para investigar tesselações com polígonos regulares. O ambiente permite formular conjecturas, testar códigos de vértices, visualizar encaixes, analisar somas angulares e diferenciar compatibilidade local de ladrilhamentos uniformes clássicos.

1 Defina a hipótese

⌨ Digitar uma configuração

Informe o código usando vírgulas, espaços ou hífens. Exemplos: (3, 6, 3, 6) ou 4-8-8.

Critério central: os ângulos internos que se encontram em um mesmo vértice precisam somar exatamente 360°. A plataforma diferencia uma compatibilidade apenas local de um ladrilhamento uniforme clássico.
Uso pedagógico sugerido: peça aos estudantes que formulem uma hipótese, testem o código do vértice, registrem a soma angular e justifiquem se há lacuna, sobreposição ou encaixe exato.

2 Fundamentação para a investigação

Ângulo interno αₙ = ((n − 2) · 180°) / n

A medida do ângulo interno depende da quantidade de lados do polígono regular.

Condição no vértice α₁ + α₂ + ... + αₖ = 360°

Soma menor produz lacuna; soma maior produz sobreposição.

Condição local e global

Fechar um vértice é necessário, mas o padrão também precisa repetir-se de forma coerente nos demais vértices do plano.
